🎨 Javier Revelo — Artista Visual & Escultor

Bienvenido al repositorio oficial del sitio web de Javier Revelo, artista visual y escultor.
Este proyecto presenta una experiencia digital centrada en la exploración de la forma, la emoción y la condición humana a través del arte.

🧩 Sobre el proyecto

Este sitio web funciona como portafolio artístico, donde se exhiben obras visuales y escultóricas mediante una interfaz moderna, minimalista y centrada en la estética.

El objetivo es transmitir no solo imágenes, sino concepto, identidad y narrativa visual.

✨ Características
🎭 Galería de obras (escultura y arte visual)
🧱 Diseño limpio y enfocado en el contenido
⚡ Componentes reutilizables con JavaScript (Web Components)
📱 Diseño responsive
🎬 Posibilidad de integrar animaciones y storytelling visual
🛠️ Tecnologías utilizadas
HTML5
CSS3
JavaScript (Web Components)

🎯 Objetivo artístico

El proyecto busca representar la relación entre el cuerpo, la materia y la emoción.
Cada obra refleja una narrativa introspectiva donde la forma se convierte en lenguaje.
